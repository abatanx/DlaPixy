# プレビュー、参照、アニメーション

## ホバー情報
- キャンバス下の 1 行で、ホバー中ピクセルの次を表示する:
  - `x,y`
  - `RGBA`
  - `#RRGGBBAA`
  - `HSVA`
  - パレット一致時の index / caption
- カーソルがキャンバス外へ出たら表示を消す。

## 参照ライン
- キャンバスピクセルまたはパレット色にホバー中に `F` で参照ラインを追加する。
- ホバー中色がパレット登録済みなら、その色も選択する。
- 参照ラインは、元色が後で変わってもパレットとの紐付きを保って更新する。
- 同じ座標で `F` を繰り返した場合:
  - 同色なら無視
  - 色が変わっていれば上書き
- 参照ラインは drag-and-drop で並び替えできる。
- 番号付け:
  - 上から `1..9`
  - 10 行目以降は `-`
- `1..9` で対応色を選択できる。
- 参照ラインの色スウォッチをダブルクリックするとパレット色モーダルを開く。
- 各項目には小さなコピーボタンがある。
- レイアウトは viewport 高に収め、必要に応じてキャンバス表示高を縮める。

## 1x Preview
- 大きい画像でも縮め切らず、スクロールで確認できる。
- プレビュー枠は角丸を使わず、見えているピクセル領域がクリップされないようにする。

## Tile Preview
- 1x Preview の下に表示する。
- 現在の選択範囲、または最後に有効だった選択範囲を使う。
- ピクセル編集をリアルタイム反映する。
- 親幅に合わせて自動調整し、正方形 viewport で表示する。
- サイドバーに追加ボタンを置き、`G` ショートカット案内も見えるようにしている。
- `G` で preview 専用の重ねとして登録できる。
- 最初の重ねが基準サイズになる。
- 以後の重ねは、その基準サイズへ clip または透明余白追加してから合成する。
- 登録済み重ね:
  - 元のキャンバス矩形にリンクし続ける
  - 編集でリアルタイム更新する
  - drag-and-drop 並び替えできる
  - 個別削除できる
  - 全消去してもキャンバス本体や Undo 履歴には触れない
- 表示結果は `3x3` の繰り返し合成プレビュー。

## Animation Preview
- Tile Preview の下にある。
- `T` または右ツールバーのボタンで現在の選択範囲をフレーム追加する。
- フレーム追加時は `Animation Preview` タブへ切り替える。
- 操作:
  - play / stop
  - FPS
  - loop toggle
  - clear all
  - delete
  - 上下移動
- 操作部はコンパクトな icon-first Bootstrap ボタン。
- プレビュー領域は正方形（`1:1`）。

## Sidebar レイアウト
- `SidebarPreviewSection` と `SidebarPaletteSection` は別カードで分ける。
- Preview 部は Bootstrap 風タブで:
  - Preview
  - Tiling
  - Animation Preview
- プレビュー内部の入れ子カード感は弱めて、フラットに見せる。
